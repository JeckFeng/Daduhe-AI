## 系统表现总结

### 亮点

1. **检索-生成链路完整**: supervisor → context_resolution → call_tools → fusion → generator → citation 全部 6 个节点正常协作，端到端延迟约 7s
2. **chitchat 快捷路径正确**: 条件路由正确分流，闲聊跳过检索和 citation
3. **多轮对话指代消解**: 5 对指代词（"这本规范"、"刚才说的"、"上面提到的"等）全部正确消解
4. **零幻觉**: 检索无结果场景全部返回"未找到"，无编造现象
5. **错误处理规范**: 参数校验/异常兜底均返回正确错误码和 trace_id
6. **数值精度**: 阈值数据（0.2mm, 0.3mm, 1.0L/s, 0.1L/s, 15%, 30%, 50%, 3年, 5年, 1月31日, ±1mm, 1:3, 0.3-1.5MPa 等）全部准确复现

### 待优化项

1. **Chitchat 回答多样性**: Q8-2 "你能帮我做什么"回答为"未找到"，可在 GENERATOR_SYSTEM prompt 中增加对能力介绍类闲聊的兜底回复模板
2. **对比回答完整性**: 当知识库不包含直接对比信息时（Q3-2/Q3-3），系统诚实但回答不够丰富。需扩充知识库或增加跨 chunk 推理能力
3. **回答一致性**: 部分回答引用编号（如 [N]）有时从 [1] 开始，有时跳号——需检查 citation 节点的 dedup 逻辑
